Run real R, right here

A real R console, running in your browser

This is not a simulation of R and not a server running your code somewhere else. R itself is compiled to WebAssembly (the webR project) and executes on your machine, in this tab. Edit any cell below and press Run, or Cmd/Ctrl + Enter. The first cell you run takes a few seconds while the R runtime downloads; every cell after that is instant.

3A real plot

Base R graphics render to a real bitmap and come back as an image, not a canned picture. Change the distribution or the sample size and rerun to get a new one.

4Open cell

Blank. Whatever you write here runs the same way. Base R only, matching this site's convention: no packages beyond what ships with R itself, so nothing installs at runtime.

→Two honest limits

Two things do not work in this mode, the tradeoff for running with zero server and zero account: a running cell cannot be interrupted mid-execution (an infinite loop will hang the cell, reload the page to recover), and functions that pause for typed input, readline(), menu(), browser(), are not supported. Everything else in base R, including stats and graphics, runs for real.
