2Why not a QQ plot

A QQ plot tells you the shape of a distribution, whether it is normal, skewed, or heavy-tailed. It does not tell you where one specific value sits. The extremeness of a value is simply how much of the data falls at or below it, its percentile on that group's own empirical distribution, reported with a confidence interval because it is a proportion. That interval is a Wilson interval, which behaves near 0 and 1 where a value can be genuinely rare.

3The consequence

Because the percentile is channel-specific, so is any decision built on it. A z-score would hide this: standardizing gives one number, but a fixed z maps to different percentiles under different shapes, so it is not comparable across channels. Report the percentile on each channel's own distribution, and let "top 5%" mean the same thing everywhere.
