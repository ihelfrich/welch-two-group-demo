Interactive teaching notes

How to think about a skewed, multi-channel score

Short interactive pieces on the questions that come up when a heavy-tailed volume measure is split across channels: how to compare groups, how to judge whether a value is extreme, and how to set a threshold that means the same thing everywhere. Every page runs on synthetic data generated in your browser. No real data is used.

The core idea

One threshold, four policies

A single cutoff flags a tenth of one channel and two thirds of another. Drag the threshold and watch a shared rule fall apart across channels.

 Extremeness

How extreme is a value?

The answer is its percentile, and it depends entirely on the channel. A value of 3 is the 90th percentile in one and the 30th in another.

 Comparing two groups

Regression, Welch, and the p-value that lies

b1 is the difference in means. The default significance test beside it is a foregone conclusion at large n, and its standard error assumes equal variance.

 Diagnostics

How to read a QQ plot

Toggle the shape of a distribution and watch the QQ plot bend. It is a shape check, not a way to locate where a single value sits.

The thread through all four: name the decision first, then the quantity that decision needs, then the tool. A test statistic is rarely that quantity.
